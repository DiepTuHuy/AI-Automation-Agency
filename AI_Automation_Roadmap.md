# AI Automation Learning Roadmap

## Mục tiêu

-   Xây AI Automation Business.
-   Đạt năng lực AI Engineer thực chiến.
-   Có 10 demo, khách hàng đầu tiên và hướng tới 100 triệu trước tốt
    nghiệp.

# Giai đoạn 1 -- LLM Fundamentals (2 tuần)

## Học

-   LLM là gì, giới hạn của LLM
-   Tokens, Context Window
-   Temperature, Top-p
-   System/User/Assistant Prompt
-   Structured Output (JSON)
-   Function Calling (khái niệm)

## Kết quả

-   Giải thích được LLM bằng ngôn ngữ của mình.
-   Viết được system prompt rõ ràng.

## Demo

-   AI trợ lý trả lời theo vai trò cố định.

## Câu hỏi đánh giá

1.  Vì sao System Prompt quan trọng hơn User Prompt?
2.  Temperature = 0 và 1 khác nhau khi nào?
3.  Context Window ảnh hưởng gì?
4.  Khi nào nên dùng Structured Output?
5.  Hãy thiết kế prompt cho trợ lý CSKH.
